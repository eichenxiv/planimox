
---

### 새로 만들기
현재 열려있는 탭을 모두 삭제하고 페이지를 전체 초기화합니다.

### 열기
.xivtl 파일을 불러옵니다.

### 저장하기
현재 탭의 내용을 .xivtl 파일로 저장합니다.

### 가져오기
* 타임라인 가져오기 : "시간 스킬명" 순서대로 입력한 여러 행의 텍스트 데이터를 읽어옵니다.
* 프리셋 : 사이트 내 저장 된 미리 정의 된 레이드 데이터를 불러옵니다.

### 시뮬레이션
* 현재 생성 된 타임라인을 시간 흐름에 따라 강조하며 볼 수 있습니다.
* 5초(기본값) / 16초 / 21초 의 초읽기 기능이 제공됩니다.

---

### 사용 방법
1. 타임라인 추가 또는 가져오기 기능을 통해 레이드 타임라인을 구성합니다.
2. 파티 직업을 설정합니다.
3. 직업 별 데미지 경감 기술, 회복 기술을 선택할 수 있으며 직접 텍스트 입력도 가능합니다.
4. 데미지 경감 기술, 회복 기술의 경우 선택한 시점으로부터 쿨다운 시간 동안에는 지정이 불가능합니다.
